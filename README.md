# 🌦️ Weather App - Real-Time Weather Forecast

![Weather App](https://user-images.githubusercontent.com/12345678/weather-app-banner.png)

> A modern weather application with a sleek UI and real-time data. 🌍✨

## 🚀 Features

- ✅ **Modern and minimalistic design** with CustomTkinter  
- ✅ **5-day weather forecast** with maximum and minimum temperatures  
- ✅ **Animated weather icons** (sun, clouds, rain)  
- ✅ **Automatic or manual location search**  
- ✅ **Favorite cities list**  
- ✅ **Dark mode with stylish colors**

## 🔧 Installation & Usage

### 1️⃣ **Clone the repository**
```sh
git clone https://github.com/leapetrini/weather-app.git
cd weather-app
```

### 2️⃣ **Install dependencies**
```sh
pip install -r requirements.txt
```

### 3️⃣ **Run the app**
```sh
python clima.py
```

## 🛠️ **Technologies Used**

- 🐍 **Python**
